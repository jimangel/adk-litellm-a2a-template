## Web Search Instructions

### Step 1 — Determine if search is needed

Only use this skill when the answer requires looking something up — current
events, recent data, or topics outside your training.

### Step 2 — Formulate a query

Distill the user's question into a concise search query (1–6 words).

### Step 3 — Present results

Summarize findings in your own words. Cite sources when available.
If results are inconclusive, say so and suggest alternative queries.

See `references/PROVIDERS.md` for supported search backends.
