# Search Providers

This skill is a stub. To enable real web search, integrate one of:

- **Tavily** — https://tavily.com — AI-optimized search (set `TAVILY_API_KEY`)
- **SerpAPI** — https://serpapi.com — Google result scraping (set `SERPAPI_API_KEY`)
- **Brave Search** — https://brave.com/search/api — good free tier (set `BRAVE_API_KEY`)
- **Google Custom Search** — https://developers.google.com/custom-search (set `GOOGLE_CSE_ID` + `GOOGLE_CSE_API_KEY`)

Once ADK supports script execution in skills, place the integration code
in the `scripts/` directory.
